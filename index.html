About

Hello, my name is David and I am a full stack web developer. I started out my professional career in technology sales and spent time in both customer facing, and technical pre-sales roles. Over the 3 year period of being in tech-sales, I found myself gravitating towards the technical side of the industry. After moving to Southern California in in 2014, I found out about LearningFuze web development 12 week-immersion boot camp and decided to to make my passion for coding a career

Every project in my portfolio has been redone several times, using new languages and techniques, to refactor and optimize code as well. My final Project, apartmentShark, which is live and in beta, is done entirely in AngularJs. This project has allowed me to dive deeper into Angular's feature-set, understanding how to better utilize dependency injection, as in using Angular Routes, ng-Animate, as well as learning more about custom directives, and to ultimately get a better handle at how to optimize and better structure my codebase for easier debugging. It's built on a PHP and MySQL backend. Since then, I've learned various technologies from Laravel to Node/Express.js, and enjoy learning and working with new technologies that speed up development.
